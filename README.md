# Qt_Government_data_dashboard
# 政府公開資料儀表板
NFU CSIE 視窗程式專題
# 環境設定
1. 安裝Qt6.6或以上版本
2. 複製路徑 Qt安裝資料夾\6.6.1\mingw_64\bin
3. 點擊win11開始鍵，搜尋「編輯系統環境變數」
4. 點擊「環境變數」
5. 系統變數選擇Path，並點選「編輯」
6. 點選「新增」，並貼上路徑
